Password Generator CLI App

Overview

This is a simple command-line application that generates secure passwords based on user preferences such as length, inclusion of digits, and special characters.


#Features

* Generate random passwords
* Customize password length
* Option to include numbers and symbols
* Simple and fast CLI interface

# Technologies Used

* Python
* Built-in libraries: random, string

# How to Run

1. Make sure Python is installed
2. Open terminal in project folder
3. Run:
   python main.py

## Example Usage

=== Password Generator ===
Enter password length: 10
Include numbers? (y/n): y
Include symbols? (y/n): y

Generated Password:
aB3@kL9#pQ
